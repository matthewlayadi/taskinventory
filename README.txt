TUTORIAL INSTALASI SERVER (BACKEND)

1. Pastikan komputer sudah terinstall Node.js.
2. Buka terminal/CMD dan arahkan ke folder ini.
3. Ketik perintah: "npm install" (untuk download library yang dibutuhkan).
4. Ketik perintah: "node index.js" untuk menyalakan server.
5. Server berjalan di http://localhost:3000.
